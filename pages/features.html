GeneX Features in Nutshell
GeneX is an online platform for analysing and visualizing the gene population, gene flow and local adaptation patterns of different species.


Features





Gene Flow Analysis


This section depicts the probability gene flow over the time.






Migration Based Analysis


Migration analysis section maps the migration patterns with the gene flow patterns.





Climate Based Analysis


In this section, we try to map the climate changes with the gene flow patterns.






Human Population Based Analysis


Population analysis provides the analysis on the impact of human population growth on gene flow.






Cause Analysis


Cause Analysis is powered by the social networking based crowdsourcing. Through the cause analysis, we try to identify the possible causes for different situations for the given species with time.
